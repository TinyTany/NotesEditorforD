・はじめに(2017/11/24更新)
NotesEditorforDは、kb10uy氏の某音ゲーシミュレーターSeaurchin用の譜面製作支援ソフトです。
視覚的に譜面を作成でき、作成した譜面はsus形式でエクスポートできます。
また、dymsという独自の形式で譜面を保存し、再度エディタで読み込むことができます

・起動方法
NotesEditorforD.exeをダブルクリックします。

・使い方
画面左に並んでいるパネルから配置したいノーツを選び、譜面上でクリックすることでノーツを配置できます。
配置したノーツを削除したい場合は、譜面上部のラジオボタンから"Delete"を選択し、削除したいノーツをクリックします。
再度別のノーツを追加したいときは同ラジオボタンから"Add"を選択してください。
ノーツパネル上のバーを操作することでノーツの大きさが変わります。
Airノーツに関してはノーツパネル上のラジオボタンを選択することで左Air，真上Air，右Airを選択することができます。
Hold,Slide，AirLineに関しては，譜面上で一度クリックすると支点の位置が決まり，そのままドラッグすることで終点の位置を決定できます。
また、配置されたHold,Slide,AirLineの終点の上でそれぞれ同じ種類大きさのノーツでクリックすることで、Lineを継ぎ足すことができます。
継ぎ足した際に残る中継点が不要な場合は削除しても構いません。
※追記(2017/11/19)
また、1譜面を越える長さのロングノーツを配置したい場合は、譜面の上端までロングノーツを配置した後、次の譜面の下端からロングノーツを継ぎ足してください。
※追記(2017/11/20)
何本かのSlideが完全に重なるようなノーツの置き方をする際は、Slideノーツを並列してではなく1本づつ確実に置いてください。
ある1本のSlideノーツの始点と終点がどこなのかを意識してください。
中継点までいくつかのSlideノーツを置き、さらにその中継点からSlideを継ぎ足すような置き方をすると正しくエクスポートされない場合があります。
この問題については今後修正する予定です。
※追記(2017/12/3更新)
Edit機能は実装されていませんのでご了承ください
Edit機能に関しては各所からコピペができなくて譜面作成がめんどくさいとの声が上がっているので今後実装していきたいと思います
エクスポート時に曲やジャケット画像を選択している場合、それらもsusファイルと同様エクスポートされるようになりました
また、曲タイトルに、ファイル・パスでは入力を禁止されている半角文字(\/:*?"<>|)を使用するとエクスポート時にエラーが出ますので、それらの
文字を使用したい時は半角ではなく全角にするなどで対処してください
また、オフセットの設定はエクスポートされたsusファイルをテキストファイルで開き、Seaurchin側で譜面を再生させながら調節するとやりやすいと思います
エクスポート画面で不明な点がある場合はkb10uy氏のsusファイルの仕様書(https://github.com/kb10uy/Seaurchin/wiki/Make-Score-Sus)を読んでください
また、BPMは仕様として小数第一位まで対応しています。

・ショートカットについて(2017/12/3更新)
ショートカットキーを追加したので以下に示します
ファイルメニュー関連
Ctrl+N　新規作成
Ctrl+O　開く
Ctrl+Shift+S　名前を付けて保存
Ctrl+S　上書き保存
Ctrl+X　エクスポート
Ctrl+Q　終了
譜面操作関連
Shift+A　Addモード
Shift+E　Editモード(未実装)
Shift+D　Deleteモード
B　分数を増加(最大時は4に戻る)
Shift+B　分数を減少(最小時は192になる)
G　グリッド数を増加(最大は4に戻る)
Shift+G　グリッド数を減少(最小時は16になる)
F1　Tapを選択
F2　ExTapを選択
F3　Flickを選択
F4　HellTapを選択
F5　Holdを選択
F6　Slideを選択
F7　AirUpを選択
F8　AirDownを選択
F9　AirLineを選択
,　選択されているノーツのサイズを1減少
.　選択されているノーツのサイズを1増加
L　Airの向きを左に
C　Airの向きを中央に
R　Airの向きを右に
